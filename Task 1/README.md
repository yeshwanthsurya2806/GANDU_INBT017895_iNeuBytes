## 📌 Project Overview

The **Healthcare Landing Page** is a modern, responsive, and user-friendly website developed as part of the **iNeuBytes Web Development Internship (Task 1)**.

The website provides an attractive interface for a healthcare organization, allowing users to explore medical services, meet doctors, learn about the hospital, and book appointments through a validated enquiry form.

---

## 🎯 Objectives

- Build a modern healthcare landing page.
- Practice responsive web design.
- Implement client-side form validation.
- Enhance user experience using JavaScript.
- Learn Git and GitHub workflow.
